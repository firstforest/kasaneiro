# パラメータリファレンス

全シミュレーションパラメータ(`SimParams`、[src/sim/mod.rs](../src/sim/mod.rs))と顔料個性の意味・既定値・UI 範囲の一覧。**定義の正典はコード側のフィールドコメント**であり、本書は一覧性のためのリファレンス。プリセットは `assets/presets/*.json`(serde default で古い JSON も読める)。

最終更新: 2026-07-05(M4 進行中の時点)

## 1. ブラシ

| パラメータ | 既定値 | UI 範囲 | 意味 |
|---|---|---|---|
| `brush_radius` | 16.0 | 1〜64 | ブラシ半径(テクセル単位) |
| `brush_water` | 0.5 | 0〜2 | 1 splat あたりに置く水量 |
| `brush_velocity` | 0.5 | 0〜2 | splat 時にストローク方向へ与える初速 |
| `brush_pigment` | 0.11 | 0〜1 | 1 splat あたりに置く顔料量(0 = 水だけのブラシ) |
| `brush_channel` | 0 | 色見本ボタン | 注入する顔料スロット(0〜3 = PIGMENTS と対応) |
| `tool` | 0 | ツールボタン | 0=描画 / 1=リフト(削り) / 2=消去 / 3=水筆 / 4=ならし(splat.wgsl が分岐) |
| `lift_strength` | 0.3 | 0〜1 | リフト(tool=1)の基準強度。実効値は顔料の ω と紙ハイト(凸部ほど剥がれる)で変調 |
| `water_lift` | 0.4 | 0〜1 | 水筆(tool=3)の均し強度: ブラシ下の顔料を近傍平均(半径2固定)へ寄せる緩和率。質量保存の箱ぼかしなので濃くならない |
| `smear_rate` | 0.35 | 0〜1 | ならし(tool=4)の均し強度: 総顔料をブラシスケールの近傍(`clamp(半径/4, 2, 10)`)へ均す緩和率。濃い山を周囲へ均一に伸ばす |

## 2. 筆圧(M1.5)

実効値 = 基準値 × `mix(1, 筆圧^γ, 効き)`。マウス(筆圧 1.0)では常に基準値のまま。

| パラメータ | 既定値 | UI 範囲 | 意味 |
|---|---|---|---|
| `pressure_radius` | 0.6 | 0〜1 | 筆圧→半径の効き(0=筆圧無効 / 1=完全比例) |
| `pressure_water` | 0.3 | 0〜1 | 筆圧→水量の効き |
| `pressure_pigment` | 0.7 | 0〜1 | 筆圧→顔料量の効き |
| `pressure_gamma` | 1.0 | 0.25〜4 | 応答カーブ γ(筆圧^γ)。1=線形、>1 で軽いタッチがより細く弱くなる |

## 3. 水の流体(velocity / relax / advect)

| パラメータ | 既定値 | UI 範囲 | 意味 |
|---|---|---|---|
| `dt` | 0.5 | 0.05〜1 | シミュレーションの時間刻み |
| `accel` | 1.0 | 0〜4 | 移流強度: 水面勾配 → 加速度の係数。水面 = 水深 + `paper_amp`×紙ハイト |
| `damping` | 0.05 | 0〜0.5 | 速度減衰(粘性の代用。1ステップあたりの減衰率) |
| `xi` | 0.1 | 0〜0.5 | 発散緩和の係数 ξ(δ = −ξ·div。Curtis 1997 の既定は 0.1) |
| `relax_iters` | 16 | 1〜50 | 発散緩和の反復回数/ステップ(安全弁の上限 64) |
| `vel_max` | 1.0 | 0.1〜2 | 速度上限(セル/ステップ)。CFL 的制約として 1.0 以下を推奨 |
| `wet_expand` | 0.0 | 0〜0.5 | にじみ拡張率: 乾いたセルのマスクが濡れた隣の水量に比例して育つ速さ(0.5 超で濡れに昇格)。0 = 筆が通った領域に固定 |

## 4. 顔料の輸送(diffuse / transfer)

| パラメータ | 既定値 | UI 範囲 | 意味 |
|---|---|---|---|
| `pigment_diffuse` | 0.15 | 0〜1 | 顔料拡散率(1反復あたり。安定条件のため WGSL 側で 0.2 に制限) |
| `diffuse_iters` | 4 | 0〜32 | 拡散の反復回数/ステップ。実効的なにじみ速度 = 拡散率 × 反復回数。速いにじみはこちらで稼ぐ |
| `deposit_rate` | 0.05 | 0〜0.5 | 吸着率: 浮遊顔料が紙へ沈着する速さ(水が少ないほど強い)。顔料の ρ と粒状ゲートで変調 |
| `lift_rate` | 0.02 | 0〜0.5 | 脱着率: 沈着顔料が水に浮き上がる速さ(水が多いほど強い)。顔料の (1−ω) で変調 |
| `evap_rate` | 0.005 | 0〜0.05 | 蒸発率: 濡れ領域の水が 1 ステップに減る量 |

transfer.wgsl の実効式(per-channel、M3):

```
吸着 = deposit_rate × ρ × dt × (2−水量) × 粒状ゲート(paper_gran×γ)
脱着 = lift_rate × dt × 水量 × (1−ω)
```

## 5. 紙(paper.rs / 3箇所適用)

紙ハイトは起動時に CPU 生成する静的テクスチャ(値ノイズ3成分: 低周波の沈殿ムラ+高周波の紙目+横方向の繊維ストリーク)。

| パラメータ | 既定値 | UI 範囲 | 意味 |
|---|---|---|---|
| `paper_amp` | 0.3 | 0〜1 | 紙ハイトの振幅: 水面勾配に足す凹凸の強さ(水が谷へ流れる → ストリーク) |
| `paper_gran` | 0.4 | 0〜1 | 粒状化: 紙の凹部で吸着が強まる度合い(顔料の γ と乗算。0=紙目の影響なし) |
| `paper_wet` | 0.5 | 0〜1 | にじみ拡張の紙目変調: 濡れ前線が谷を選んで進む度合い(縁が不規則に。0=一様) |

## 6. エッジダークニング

2方式あり、現行は乾燥時方式(`dry_edge`)。FlowOutward は既定オフで残置(status.md M1d 参照)。

| パラメータ | 既定値 | UI 範囲 | 意味 |
|---|---|---|---|
| `edge_eta` | 0.0 | 0〜0.2 | FlowOutward の η: 濡れ領域の縁ほど水を除去する強さ。**既定 0 = パスごと省略(先送り)** |
| `edge_radius` | 4 | 1〜8 | 縁バンド幅(ぼかし半径、テクセル)。FlowOutward と `dry_edge` の縁バンドで共用 |

## 7. 乾燥(M2、bake / rewet)

| パラメータ | 既定値 | UI 範囲 | 意味 |
|---|---|---|---|
| `dry_shift` | 1.0 | 0〜1.5 | 焼き込み時に顔料濃度へ掛ける係数。**既定 1.0 = 乾いても濃度を保つ**(アナログ調に薄くしたいときだけ <1 に下げる) |
| `dry_gran` | 0.0 | 0〜1 | 焼き込み時の粒状感ゲート: 紙の凹部で濃く/凸部で薄く定着(描画中の `paper_gran` とは独立) |
| `dry_edge` | 0.4 | 0〜2 | 焼き込み時のエッジダークニング: 顔料被覆率の縁バンド(ぼかし残差)で濃度を増す強さ。効きは弱め(既知、status.md 参照) |
| `rewet_water` | 0.5 | 0〜2 | Wet the Layer(全面再湿潤)で足す水量 |

焼き込み式: `焼き込み濃度 = (浮遊+沈着) × dry_shift × 粒状感ゲート × (1 + dry_edge×縁バンド)`

## 8. 表示・合成(display.wgsl)

| パラメータ | 既定値 | UI 範囲 | 意味 |
|---|---|---|---|
| `pigment_density` | 1.0 | 0.5〜10 | 発色の濃さ: 顔料濃度 → 被覆率 `1−exp(−density·総濃度)` の係数 |
| `compose_mode` | 1 | 切替ボタン | レイヤー合成: 0=multiply(乗算。重ねるほど暗い安価な近似)/ 1=KM(Kubelka-Munk の R/T 光学合成。薄い層ほど下が透けるグレーズ) |
| `display_mode` | 0 | コンボボックス | デバッグ表示(H4): 0=通常 / 1=水量ヒートマップ / 2=速度場(色相=方向)/ 3=湿りオーバーレイ(濡れ=青)/ 4=浮遊顔料 / 5=沈着顔料 / 6=紙ハイト(白=山) |
| `display_gain` | 1.0 | 0.1〜10 | デバッグ表示の輝度スケール |

## 9. 顔料個性 ρ/ω/γ(M3、[src/pigment.rs](../src/pigment.rs))

SimParams とは別に、顔料ごとの物性3値を uniform(binding 9)で渡す。現状はコード内の const(M5 で編集 UI 化予定)。

| 物性 | 範囲 | 意味 |
|---|---|---|
| 密度 ρ (`density`) | >0 | 沈着(吸着)の速さ。重い顔料ほど早く沈む(transfer の吸着を倍率変調) |
| ステイニング ω (`staining`) | 0〜1 | 剥がれにくさ。高いほど脱着・リフトが (1−ω) で抑えられ紙に色が残る(ステイン床) |
| 粒状感 γ (`granulation`) | 0〜1 | 紙ハイトへの反応。高いほど凹部(谷)に沈着し凸部で剥がれる = 粒状化 |

現行パレット(mixbox 推奨顔料色、対比を付けた代表値):

| スロット | 顔料 | sRGB | ρ | ω | γ | 性格 |
|---|---|---|---|---|---|---|
| 0 | ハンザイエロー | 252,211,0 | 1.0 | 0.5 | 0.1 | 半透明・中ステイニング・ほぼ非粒状 |
| 1 | フタロブルー | 13,27,68 | 0.8 | 0.9 | 0.05 | 微細粒子。強ステイニング(剥がれない)・非粒状・軽い |
| 2 | キナクリドンマゼンタ | 128,2,46 | 0.9 | 0.85 | 0.05 | 透明・強ステイニング・非粒状 |
| 3 | バーントシェンナ | 123,72,0 | 1.3 | 0.2 | 0.8 | アース系。重く早く沈着・低ステイニング(よく剥がれる)・強粒状(紙目に溜まる) |

## 10. パラメータを追加するときの手順(H2 の定型作業)

1. [src/sim/mod.rs](../src/sim/mod.rs) の `SimParams` にフィールド1行(末尾の `_pad` を置き換える。16 バイト整列を保つ)+ `Default` に既定値
2. [src/app.rs](../src/app.rs) にスライダー1行
3. [assets/shaders/common.wgsl](../assets/shaders/common.wgsl) の `struct SimParams` に1行(Rust 側と同レイアウト)

serde(default) により古いプリセット JSON はそのまま読める。`_pad` は serde(skip) で JSON に出さない。
